# RSS 当日资讯 — 参考

## 默认收录方向 `feeds.json`

## 每个数据源合并去重后最多 5 条；每个「类型」最多 15 条；优先抓取中文环境文章

## 把 标题，摘要等都翻译成中文

## 常见问题

| 现象 | 处理 |
|------|------|
| HTTP 403 | 更换 User-Agent 已在脚本中设置；仍失败则换该站的其它 RSS 地址或暂缓 |
| 条目为空 | 当日该源可能无更新；检查日期是否应为昨天；或 feed 仅含少量条目 |
| 时间错位 | 脚本按 `Asia/Shanghai` 日历日过滤；跨国用户需说明时区 |

## Hugo

`date` 建议使用当日早间固定时间 + `+08:00`，与示例帖一致；`draft` 默认 `true`，发布前改为 `false`。
